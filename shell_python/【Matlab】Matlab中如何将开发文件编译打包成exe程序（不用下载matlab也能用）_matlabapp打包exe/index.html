这就需要App编译打包工具了，往下看：

1.这个编译工具也已经集成在内部App应用中了，可以直接打开它；


2.界面如下：添加文件和相关描述（与上一期打包内部App类似）；


3.等到打包完成（打包外部可执行程序时间稍微长一点）；


4.在生成的这个文件夹中可以看到exe程序；


5.打开exe，与在matlab中运行的结果一致，完成。
