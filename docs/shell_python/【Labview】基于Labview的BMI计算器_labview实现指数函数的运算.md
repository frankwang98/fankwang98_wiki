在做之前，百度了一下BMI值的计算方法，主要还是**根据国人的习惯，输入身高/cm和体重/kg，然后输出BMI的计算结果和一个判断（是正常体重，还是偏瘦，过重）**，基本思想就是这样，当然还是比较简单的，主要是熟悉一下Labview的操作，适合入门者做一下。
