### Blender介绍


Blender是一款**免费**开源的三维图形图像软件，提供从建模、动画、材质、渲染、到音频处理、视频剪辑等一系列影视制作的全流程解决方案。


Blender的特点：


* 小巧
* 免费
* 全流程


Blender的用处：


* 建模并导出到游戏引擎（偏游戏开发）
* 全流程制作CG短片（偏影视制作）

### Blender界面


Blender软件界面 如下：


![在这里插入图片描述](https://img-blog.csdnimg.cn/191631ebda6741bdbc1ab3e34a1e5a81.png)


坐标系：


* X轴：红色
* Y轴：绿色
* Z轴：垂直向上


快捷键：


* 旋转：鼠标中键拖动（Alt+R旋转归零）
* 缩放：鼠标滚轮（Alt+S缩放归零）
* 平移：shift+鼠标中键（Alt+G坐标归零）
* 复制：Ctrl+C/V & Shift+D


视图操作（小键盘）：


* 数字1：正视图
* 数字3：右视图
* 数字7：顶视图
* 四格视图同时显示：Ctrl+Alt+Q
